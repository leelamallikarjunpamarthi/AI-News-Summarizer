## AI-News-Summarizer
AI News Summarizer is an AI-powered web application that summarizes news articles from uploaded documents, PDFs, and text files. It highlights key points and includes an Ask AI chat feature to interact with the documents and get quick answers
---

## Project Overview
This project implements an AI-powered News Summarizer and Document Intelligence System.

Users can upload documents such as PDFs, text files, and news articles. The system stores the documents, processes them using AI, and generates summarized content with highlighted key points. Additionally, the platform includes an Ask AI module that allows users to chat with AI about uploaded documents.

Traditional document reading and news analysis take a lot of time. This system automates reading, summarizing, and extracting important insights using Artificial Intelligence.

The platform transforms large documents into short, meaningful summaries and allows users to interact with the documents using AI chat.

---

## Problem Statement
Reading full news articles and large documents is time-consuming. Users often want:

- Quick summaries
- Important highlights
- Key insights
- Ability to ask questions about documents
- Smart document storage and retrieval

Traditional systems only store documents but do not understand the content.

Our system uses Artificial Intelligence to read, understand, summarize, and answer questions from documents.

---

## Key Features
- AI-powered news and document summarization
- Upload PDF and text documents
- Extract key highlights and important points
- Ask AI about uploaded documents
- Document storage using MongoDB
- Secure authentication and user login
- REST API backend (Node.js + Express)
- React + Tailwind frontend
- OpenAI API integration for summarization and chat
- Responsive modern UI
